AlgoSuite
Small web app with 2 algorithms:
-	Snail Matrix → reads a square matrix in spiral order.
-	Consecutive Numbers → checks if a digit string can form numbers that increase by +1.
Built with HTML, CSS, JavaScript only.
How to use :
Open index.html in your browser.
Choose a tab:
Snail Matrix → select size and click Generate Spiral.
Consecutive → enter a number string and click Check Validity.
No installation needed.
Example
Snail (n = 3)
Result:
1 2 3
4 5 6
7 8 9

-1 → 2 → 3 → 6 → 9 → 8 → 7 → 4 → 5
 
Consecutive
Input:
