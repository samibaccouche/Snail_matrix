# 🚀 AlgoSuite

AlgoSuite is a lightweight web application that demonstrates two classic algorithmic problems using pure **HTML, CSS, and JavaScript**.

It is designed to be simple, interactive, and easy to run without any installation.

---

## ✨ Features

### 🌀 Snail Matrix

* Generates an **n × n matrix**
* Traverses it in **spiral (snail) order**
* Visual and easy to understand output

### 🔢 Consecutive Numbers

* Validates whether a numeric string can be split into a sequence of numbers
* Each number must increment by **+1**
* Supports multi-digit transitions (e.g., 99 → 100)

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling
* **JavaScript** – Logic & Interactivity

---

## ▶️ How to Use

1. Download or clone the repository
2. Open `index.html` in your browser
3. Choose one of the available features:

### Snail Matrix

* Select the matrix size (n)
* Click **"Generate Spiral"**
* View the matrix and its spiral traversal

### Consecutive Numbers

* Enter a numeric string
* Click **"Check Validity"**
* See whether the sequence is valid or not

## 📌 Examples

### 🌀 Snail Matrix (n = 3)

**Matrix:**

```
1 2 3
4 5 6
7 8 9
```

**Spiral Order:**

```
1 → 2 → 3 → 6 → 9 → 8 → 7 → 4 → 5
```

---

### 🔢 Consecutive Numbers

**Input:**

```
979899100101
```

**Output:**

```
Valid → 97 → 98 → 99 → 100 → 101
```

---

### ❌ Invalid Example

**Input:**

```
1235
```

**Output:**

```
Invalid sequence
```
